Proposito del sitio:

Este sitio esta desarrollado con el fin de enseñar da una
forma sencilla algunos conceptos básicos del diseño web como
lo son las etiquetas básicas en HTML y CSS GRID, se explican
con funcionan estos elementos y en que circunstancias se
pueden utilizar.

Introducción a las tematicas:

Que son las etiquetas semánticas:

Una etiqueta se califica como semántica si tiene que ver el significado,
es decir, si nos informa sobre lo que trata su contenido. Por ejemplo,
la etiqueta SECTION nos dice que contiene una sección o capítulo dentro
de la página. (Aula Clic, 2015)

CSS Grid

CSS Grid layout contiene funciones de diseño dirigidas a los desarrolladores de aplicaciones web.
El CSS grid se puede utilizar para lograr muchos diseños diferentes. También se destaca por
permitir dividir una página en áreas o regiones principales, por definir la relación en términos
de tamaño, posición y capas entre partes de un control construido a partir de primitivas HTML.
